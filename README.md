# Simple Calc

A simple Python calculator project that demonstrates basic Git and GitHub usage.

## Features
- Addition
- Subtraction
- Multiplication
- Division (with zero-check)
- Exponential

## Requirements
- Python 3.7+  
- Virtual environment (recommended)
