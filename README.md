# Simple Calc

A simple Python project that perfoms basic mathematical operations on any two given numbers

## Features
- Addition
- Subtraction
- Multiplication
- Division (with zero-check)
- Exponential

## Requirements
- Python 3.7+  
- Virtual environment (recommended)
